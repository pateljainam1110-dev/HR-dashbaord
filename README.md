# HR Analytics Dashboard 

This is an HR Analytics Dashboard built in **Power BI** to analyze employee attrition and workforce trends. The dashboard provides insights into employee count, attrition rate, age distribution, department-wise attrition, education field, and job satisfaction.

## Features
- Employee Overview
- Attrition Analysis
- Department-wise Insights
- Age Group Analysis
- Education-wise Attrition
- Interactive Filters

## Tools Used
- Power BI
- Power Query
- DAX
